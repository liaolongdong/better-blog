# 超好用的 chrome 浏览器插件推荐（外语阅读困难者福音）

安装一些工作中常用的chrome插件，可以极大的提升我们的工作效率，下面推荐几款使用体验超爽的chrome插件。

## Google 翻译

插件功能：**强烈推荐外语文档阅读困难者安装该插件**，通过该插件可以翻译成各种语言，支持整页、段落、选词翻译等。

用法：点击插件图标，输入要翻译的文本，然后点击翻译按钮即可，鼠标双击选中单词翻译，三击选中整行翻译，右键单击选中段落翻译，点击“翻译当前页”翻译整页。

**PS：使用 谷歌翻译 可能需要科学上网**

![Google 翻译](/assets/img/chrome-plugin-img/google-translate.png)

## 沉浸式翻译: 双语对照网页翻译 & PDF文档翻译

插件功能：**强烈推荐外语文档阅读困难者安装该插件**，该插件可以设置网页中英文双语阅读，翻译扩展，支持PDF翻译，双语Epub电子书制作，Youtube/Netflix/Udemy 等平台双语字幕，支持Deepl/Google等多个翻译服务，免费使用。一款免费的，好用的，没有废话的，革命性的，饱受赞誉的，AI 驱动的双语网页翻译扩展，帮助你有效地打破信息差，在手机上也可以用！

用法：点击插件图标进行相应的配置即可，免费版感觉够用了，对翻译要求比较高的可以使用付费版。

![中英双语](/assets/img/chrome-plugin-img/zh-en-translate.png)

## Web开发者助手 FeHelper

功能：该插件支持Chrome、Firefox、MS-Edge浏览器，内部工具集持续增加，目前包括 JSON自动/手动格式化、JSON内容比对、代码美化与压缩、信息编解码转换、二维码生成与解码、图片Base64编解码转换、Markdown、 网页油猴、网页取色器、脑图(Xmind)等贴心工具，甚至在目前新版本的FeHelper中，还集成了FH开发者工具， 如果你也想自己搞一个工具集成到FeHelper中，那这一定能满足到你。

用法：点击插件图标使用对应功能即可。

![前端助手](/assets/img/chrome-plugin-img/fe-helper.png)

## GoFullPage - Full Page Screen Capture

功能：该插件完整可靠地捕获当前页面的屏幕截图 - 无需请求任何额外的权限！获取当前浏览器窗口的整页屏幕截图的最简单方法，支持将结果导出为 PNG、JPEG 或各种 PDF 纸张尺寸 - 所有这些均可通过扩展程序的选项进行配置，同时也可查看截图历史记录。

用法：点击插件图标使用即可。

![网页全屏截图](/assets/img/chrome-plugin-img/full-page-screen-capture.png)

## Nimbus 截幕 & 屏幕录像机

插件功能：截屏网页，无论是网页的一部分还是整个浏览器窗口，在一个非常用户友好的界面中编辑您的屏幕截图。 录制截屏视频 - 从屏幕录制视频。

用法：在需要截图或者录屏的页面，点击插件图标使用即可。

![Nimbus 截幕 & 屏幕录像机](/assets/img/chrome-plugin-img/nimbus-screenshot.png)

## Vue.js devtools

插件功能：用于调试 Vue.js 应用程序的 Chrome 开发工具扩展。

用法：启动本地Vue项目，在chrome浏览器中打开，打开chrome开发者工具，即可看到Vue的调试工具。

![Vue.js devtools](/assets/img/chrome-plugin-img/vue-devtools.png)

## React Developer Tools

插件功能：用于调试 React.js 应用程序的 Chrome 开发工具扩展。

用法：启动本地React项目，在chrome浏览器中打开，打开chrome开发者工具，即可看到React的调试工具。

![React Developer Tools](/assets/img/chrome-plugin-img/react-developer-tools.png)

## console-importer

插件功能：安装该插件后，可以在chrome浏览器控制台安装 npm 依赖包，比如 jQuery、Vue等第三方依赖库。

用法：打开Chrome devtools控制台，可以使用名为 $i 的函数导入JavaScript和CSS资源。详细用法查看 https://github.com/pd4d10/console-importer

![console-importer](/assets/img/chrome-plugin-img/console-importer.png)

## 篡改猴

插件功能：篡改猴是一款功能强大的浏览器扩展功能，拥有超过1000万用户，可以在网站上运行用户脚本，从而提升您的浏览体验。许多中国用户称篡改猴为油猴。用户脚本是一些小型程序，可以修改页面布局、添加或删除功能以及自动执行操作，个性化您的网络体验。

### 🔑 主要功能：

🛠️ 高效管理和编辑您的用户脚本
⚡ 用两次点击迅速激活和停用脚本
🔗 使用浏览器同步和各种云存储服务（Google云端硬盘、Dropbox、OneDrive、Yandex.Disk 和 WebDAV）同步脚本
💾 通过压缩文件和/或云存储备份和恢复选项和用户脚本
🔄 自动更新脚本，实现无缝体验

![tampermonkey](/assets/img/chrome-plugin-img/tampermonkey.png)

## Proxy SwitchyOmega

插件功能：轻松快捷地管理和切换多个代理设置。

用法：点击插件图标进行相应的配置即可。

![proxy](/assets/img/chrome-plugin-img/proxy.png)

## Axure RP Extension for Chrome

插件功能：你还在使用IE或者火狐浏览器查看UE原型图吗？使用该插件允许从 Google Chrome 查看本地发布的 Axure RP 原型（HTML 文件）的扩展。 添加此扩展程序可以在 Google Chrome 中查看使用 Axure RP 创建的本地发布的原型（HTML 文件）。

用法：使用chrome浏览器直接打开Axure原型即可查看。

![Axure](/assets/img/chrome-plugin-img/Axure.png)
